# 💡〰️ `pyscopee` 〰️🚀

## 🔭 Scope(e)

`pyscopee` aims to streamline working with spectroscopy, especially detailed analysis of
spectroscopic data such as

- interferograms of Fourier Transform spectrometers 💥〰️🪞〰️👀
